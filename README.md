# FlightSuretyApp

FlightSuretyApp is a project for Udacity's Blockchain course.

## Install

To install, first clone the repo, and do:

`npm install`
`truffle compile`

## Client Side

To run test cases:

`truffle test ./test/flightSurety.js`
`truffle test ./test/oracles.js`

To use the Dapp:

`truffle migrate`
`npm run dapp`

To view Dapp:

`http://localhost:8000`

## Server Side

`npm run server`
`truffle test ./test/oracles.js`

Deploy the contents of the ./dapp folder
